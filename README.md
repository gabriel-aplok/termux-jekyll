# Jekyll Termux
I made a simple script to setup jekyll on termux to help my friend keep your blog hosted jekyll on mobile.

## 1. Step

```shell
apt install git
```

## 2. Step

```shell
git clone git@github.com:gabriel-aplok/termux-jekyll.git
```

## 3. Step

```shell
cd jekyll-termux 
```

## 4. Step

```shell
./setup.sh
```
